# JR_Module1_bot

## Опис

Бот є посередником між користувачем та ШІ.
Задачі, які може виконувати бот:

- надіслати випадковий факт
- відповісти на питання користувача
- імітувати розмову з відомою особистістю
- перевірити знання користувача по деяким наукам та технологіям
- провести бесіду голосом з ШІ GPT 🎙
- порекомендувати фільмі, книгу, музику

## Установка

pip install -r requirements.txt

## Налаштування

Скопиюй .env.example, зміни назву файлу на .env и заповни змінні своїми значеннями

## Запуск

python bot.py
